# Birthday Wisher

## Objetivo
Enviar emails de cumpleaños automáticamente usando un CSV y plantillas de carta.

## Requisitos
- Python 3.10+
- pandas
- Cuenta de email SMTP (recomendado app password)

## Archivos
- `birthdays.csv`
- `letter_templates/`
- `main.py`

## Ejecución
```bash
python main.py
```
