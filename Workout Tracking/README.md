# Workout Tracking

## Objetivo
Analizar ejercicios en lenguaje natural y guardar resultados en una hoja (Sheety).

## Requisitos
- Python 3.10+
- requests
- Nutritionix API (APP_ID, API_KEY)
- Sheety endpoint

## Configuración
Completa tus credenciales/endpoints en `main.py` o cárgalos desde variables de entorno.

## Ejecución
```bash
python main.py
```
