# Habit Traker

## Objetivo
Registrar hábitos diarios en Pixela y visualizar el progreso en un gráfico.

## Requisitos
- Python 3.10+
- requests
- Credenciales de Pixela (usuario y token)

## Configuración
Edita `main.py` con tu `USERNAME`, `TOKEN` y endpoint correcto.

## Ejecución
```bash
python main.py
```
